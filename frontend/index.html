About Makeitso

Make it so is a bare startup theme with the minimum of styling. It is set up with the assumption a modern day website is centered and full width. It is however very easy to remove the full width aspect and make a max-width centered layout like this one. Make it so is responsive and built mobile first. Any changes and additions you make should be added in a mobile first workflow. As the viewport widens with every larger sized medium, styles get applied and/or overwritten.

Not an extended HTML framework

You don't need Foundation or Bootstrap to built a sleek and efficient front-end, nor should you need Make it so for that matter. For protoyping, a framework like Foundation can be handy to speed things up, but in my opiniion you get a lot of update and syncing hassle and weight with it. Make it so strives to not give you unneeded and excessive markup, styles and classes or pre-made design patterns:

  • Normalize.css is used as default to align browsers' rendering appearrance and positioning. The rest is bare bones...
  • Need a grid in place at a certain break-point? Just declare (wrapper) widths in percentages, float them and be done with it. Not 'col-6', 'spacer-3' or 'med-12-4-1-grid_home_bloat' classes...

Use your head, your creative skills and ye'olde CSS plus good accessible HTML!
